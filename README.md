## Mobile Client

To get started run:
```
cd MobileClient
npm install
npm start
```

And open the `Expo client` in your android phone.

## Arduino

Sensors used:
- Heartbeat
- Distance
- UV
- Tilt
- Waterproof ultrasound distance
- (Optional) Breathing measurement strap

Boards used:
3 esp8266 boards

Additional accessories:
- Water bottle
- Hat

To compile code to boards use `Arduino IDE`.
